# 読書ツール — 使用方法指南

本工具是一款网页版双语阅读辅助工具，全面支持英文和日文电子书籍，通过深度集成 Google Gemini AI 技术，提供划词翻译、语法剖析、全书解析、以及防抖自动保存的沉浸式阅读笔记专属功能。

---

## 一、启动工具

### 1. 配置 API Key

在项目根目录的 `.env` 文件中填入你的 Google Gemini API Key：

```
GEMINI_API_KEY=AIzaSyxxxxxxxxxxxxxxxxxxxxxxx
```

> 如何获取 API Key：访问 Google AI Studio 创建并获取属于你的 Gemini API 密钥。

### 2. 启动服务

```bash
cd "読書ツール"
npm run dev
```

浏览器访问：**http://localhost:5173**

## 三、双轨工作流操作指南

在全新版本的 読書ツール 中，右侧工作区被切分为了极其强大的双轨 Tab。

### 核心功能区：【✨ AI アシスタント】
1. **自动语种匹配**：如果你查询日文，AI将生成详尽的日语 N3 级别以上的高阶语法与假名；如果查询英文，AI 将采用全境纯正英语（英英词典模式）输出 CET-4 级别的高阶词汇和句子搭配。
2. **文本选定联动**：用鼠标在左边书中划线选取文本，它会自动镜像填充在右侧顶部。
3. **四引擎爆发**：
   - `🔍 単語を調べる`：查询单词释义和例句。
   - `📐 文法解析`：针对长句进行语法、结构句型剥离。
   - `✨ AI 分析`：自由解析框架。支持你在【自定义 Prompt 框】里对这段话追加特殊要求（比如：“给我翻译成莎士比亚风格”）。
   - `📚 全書分析`：宏观分析引擎。不需要选中任何文本，甚至可以利用【自定义 Prompt 框】书写特定章节或者主题的挖掘命令，只要点击，它即刻通读全书并顺着你的旨意总结核心内容。

### 私人手帐区：【📝 読書ノート】
- **一书一笔记**：你在此处记录的所有感悟、总结，全部与当前左侧阅读的书籍“灵魂绑定”。
- **防抖隐式归档（Auto-save）**：你可以像使用 Obsidian 一样疯狂敲击键盘，当你停顿思考超过 1 秒钟时，系统会在后台悄无声息地进行秒速归档，保障绝不丢失数据。无需你操心去点击任何保存按钮。

## 四、本地宇宙：阅读记忆的结构化沉淀

本工具的所有查阅痕迹不仅在网页上可看，更是一座属于你的无形金矿。每一次有价值的 AI 获取，都会在你的电脑硬盘里直接生成完美的 Markdown。

**储存路径**：项目根目录下的 `/history/[书籍名称]/`

在你深度阅读一本书后，这个目录下将诞生以下排版极其出色的 Markdown 开源文档，方便你随时导入其它笔记软件：
- `Vocabulary.md` （所有查过的复杂生词集合）
- `Grammar.md` （所有难懂句风的历史破解）
- `AI_Analysis.md` (自由问答存档)
- `Book_Analysis.md` (整本书籍脉络梳理)
- `Notes.md` (你在笔记 Tab 里高强度思考的精华)

## 五、阅读器底层技术手册

### PDF 文档
- **极速串流**：抛弃了一次性读取内存卡死的痼疾，改为**按页动态渲染**。几百兆的书也能瞬间打开。
- **拖拉字取**：在渲染页面时支持隐藏 DOM 字图层交叠技术，可以直接鼠标划取。

### EPUB 文档
- **智能防呆机制**：当你刚打开一本宏大的 EPUB 时，左下角会显示 `全文抽出中...`。如果是复杂的内置加密 EPUB 并发提取文字失败，底层框架也会自我降级，为你启动强劲的错误绕道防波堤，绝不会出现整本书籍崩溃的问题。
